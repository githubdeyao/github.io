五、解决MySQL5.7忘记root密码的办法

在DOS窗口下输入net stop mysql
切换到mysql的bin目录下另开一个DOS窗口输入mysqld --skip-grant-tables
不要关闭窗口，再开一个DOS窗口先登陆：mysql -u root
输入：SET PASSWORD FOR 'root'@'localhost' = PASSWORD('newpass')

查看表结构

+——-+————-+——+—–+———+——-+
| Field | Type | Null | Key | Default | Extra |
+——-+————-+——+—–+———+——-+
| id | int(11) | YES | | NULL | |
| name | varchar(20) | YES | | NULL | |
| sex | char(1) | YES | | NULL | |
+——-+————-+——+—–+———+——-+

  • Field ： 字段名
    Type ： 类型
    Null ： 是否可以存储为NULL
    Key ：是否为主键或外键
    Default ：默认值
    Extra ：附加信息，比如：AUTO_INCREATMENT（自增长）

补充：

  • 主键： 唯一标识一条记录，不能有重复的，不允许为空；一张表中只能有一个

  • 外键：表的外键是另一表的主键, 外键可以有重复的, 可以是空值；
    一个表可以有多个外键
